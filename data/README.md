# Data Source

   This project uses the **2025 County Health Rankings & Roadmaps (CHR&R) Analytic Dataset**,
   published by the University of Wisconsin Population Health Institute with support from
   the Robert Wood Johnson Foundation.

   - **Source:** https://www.countyhealthrankings.org/health-data
   - **File used:** `analytic_data2025_v3.csv`
   - **Data dictionary:** `DataDictionary_2025.pdf` (included in this repo / also available at the source above)
   - **Coverage:** 3,204 rows (U.S. national, 51 states/DC, and 3,152 counties), 796 variables

   The raw dataset is not redistributed in this repository. To reproduce this analysis,
   download the file directly from CHR&R's Health Data portal and place it in a local
   `data/` folder before running the notebooks.
